Elementos para citações

Elemento "cite"

"Quem espera que a terra mude, ou que o tempo fique bom, nunca plantará, nem colherá nada".

Esse elemento serve para indicar a fonte de uma citação ou o título de uma obra.

Exemplo:

Como disse Albert Einstein,A imaginação é mais importante que o conhecimento.

Elemento "q"

Esse elemento q(quote/citar) serve para marcar citações CURTAS dentro de um texto sendo a "aspas".

Elemento "blockquote"

O elemento blockquote (bloco de citação longa) serve para indicar uma citação em bloco, ou seja, um trecho de texto que foi retirado de outra fonte.
