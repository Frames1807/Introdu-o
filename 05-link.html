Conhecendo o elemento de navegação

Elemento "a"

Um link é um elemento que permite navegar de uma página, documento ou recursos para outro.

No html, a gente usa os links para:

  • Abrir um site externo.
  • Levar o usuário para outra página do site.
  • Abrir aplicativos com e-mail e telefones.

Conhecendo o elemento "a"

O elemento usado para criar links é a tag "a" (anchor)

href - hyperText Reference

target - É um atributo que define onde o conteúdo do link será aberto. E nele informamos a função especifica. Veja algumas delas a seguir:

  • Blank: explicação...
  • Self:
  • parent:
  • Top:
