Conhecendo o elemento de imagens

Sobre imagens

Trabalhar com imagens no HTML envolve muito mais do que apenas colocar uma foto na pág.

Existem várias formas técnicas e recursos para exibir, organizar, otimizar e adaptar imagens para diferentes dispositivos.

Formatos mais comuns de imagens

  • JPEG/JPJ: Fotos e imagens compactadas
  • PNG: Suporta transparência
  • WEBp: Formato moderno e leve (também suporta transparência)
  • Gif: Animações simples
  • Avif: Excelente qualidade com um tamanho pequeno

Sobre o elemento img

A forma mais comum de inserir imagens no HTML é usando a tag img ( img src ="" alt="")

Exemplo prático para adicionar imagem com links externos

Trabalhando com imagens locais também com img ( img src ="" alt="")

SRC (source / fonte) - Atributo responsável pelo caminho da imagem, ou seja, define onde está a imagem

ALT (alternative/alternativo - É o texto alternativo que será apresentado caso a imagem não carregue.

O atributo alt é MUITO IMPORTANTE

  • Acessibilidade
  • Leitores de telas
  • SEO
  • Fallback

Ajustando o tamanho das imagens

Quando tratamos de tamanho, usamos o "height" dentro de "alt" na tag da img, e quando tratamos de largura, usamos o "width"

HEIGHT esse atributo serve para ajustar o tamanho da altura da imagem

WIDTH esse atributo serve para ajustar o tamanho da largura da imagem

Lembrando quê, quando falamos de imagem, também falamos de pixels
Os pixels (px) são uma unidade de medida usada para definir TAMANHOS e ESPAÇAMENTOS na interface

Trabalhando com imagens responsivas

O picture é um elemento do HTML usado para imagens responsivas e adaptativas.

Essa tag, permite o segunte:

  • Trocar a imagem conforme o tamanho da tela.
  • Melhorar a performace
  • Carregar imagens diferentes para mobile, tablet e desktop

Dentro da tag "picture" usamos o "source" e dentro dele inserimos o atributo "srcset".

srcset - Serve para oferecer múltiplas versões de uma mídia e deixar o navegar escolher a mais adequada conforme o tamanho da tela, formato suportado, largura dísponível e performace.
