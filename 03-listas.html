Etendendo as listas do HTML

A listas no HTML, servem para organizar informações relacionadas de forma estruturada e semântica.

As listas são compostas por elemento chamados de LI.

Os LI (list item/Item da lista). As listas não possuem limite de itens.

Orderede List - OL

O elemento OL serve para criarmos uma lista ordenada, ou seja, com ordem, importância.

Normalmente aparece numerada.

Preparando um bolo

  1. Separe os ingredientes
  2. Juntar cada ingrediente
  3. Levar ao forno

Veja que não podemos assar a massa, sem antes preparar os ingredientes, então, a numeração segue uma lógica de importância.

Também conseguimos usar o "type" para mudar a configuração do estilo da contagem.

  1. Comprar
  2. Organizar
  3. Assar

Além disso, pode-se implementar "start" e "reversed".

Unordered List

Serve para criar listas sem ordem específica. Normalmente aparece com bolinhas (bullets)

Exemplo rápido

  • Comprar o fermento
  • Comprar ovos
  • Comprar farinha

Veja que a ordem das compras não importa, ou seja, tanto faz iniciar comprando o fermento, ou a farinha.

Description List - DL

O elemento DL é usado para criar uma lista de definição no HTML.
Podemos usar em: Perguntas e respostas, termos...

Exemplo de aplicação
